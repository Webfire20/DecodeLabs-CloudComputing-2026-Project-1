# Caroline Dion — Design Portfolio

A modern responsive portfolio website for showcasing graphic design work, built using pure HTML, CSS, and JavaScript.

## Features

- Fully responsive layout
- Mobile-friendly navigation
- Smooth scrolling
- Modern dark monochrome aesthetic
- Portfolio showcase sections
- Cross-platform optimization
- AWS S3 static hosting compatible

## Sections

- Hero / Cover
- About Me
- Work Experience
- My Work Gallery
- Contact / Closing Section

## Tech Stack

- AWS S3
- HTML5
- CSS3
- Vanilla JavaScript
- IAM Policies
- JSON Bucket Policy

## Website
- http://dino-portfolio-2026-788884270727-ap-south-1-an.s3-website.ap-south-1.amazonaws.com/

## Key Concepts Learned
- Infrastructure as a Service (IaaS)
- Object Storage vs Compute
- Principle of Least Privilege
- 11 Nines durability (99.999999999%)
